# NOITOM Robotics 参观预约系统

## 上传方式
1. 解压本 ZIP。
2. 进入解压后的文件夹。
3. 全选所有文件。
4. 上传覆盖 GitHub 仓库根目录。
5. 点击 Commit changes。
6. 等待 GitHub Pages 更新。
